# Source validation

Publication blocks on missing coverage, empty extraction, invalid coverage, stale source weeks, and other blocking conflicts. Explicit update timestamps are source evidence; HTTP dates are deliberately not substituted. Unknown lane capacity remains `null`, is displayed as “not stated”, and is never converted into an invented lane layout.

Snapshots store requested/final URL, status, fetched time, selected response headers, content hash, HTML, and metadata. Structural signals (headings, day sections, rows and interpretation ratio) travel with extraction for comparison and future alerting.
